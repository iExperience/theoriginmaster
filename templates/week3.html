Taking HTML & CSS Farther

But what if you want animations that melt the faces off your visitors? What if you want to learn how to make super complex layouts? We've put together some resources that you can refer to as you go out and continue building things for the web.

HTML

  • HTML5 Element index

    This is just a list of all the elements available in HTML5. You might be surprised at how many there are, and how specific they get. The more accurately you can match an element to the contents it contains, the better.

  • Top 10 Reasons to Use HTML5 Right Now

    A blog post highlighting some of the main features in HTML5, and showing a lot of the powerful and new things it can do. It also has a lot of useful and interesting links about how to use HTML5.

CSS

  • Creative Button Styles

    A bunch of cool button effects that you can use in your app, use as inspiration, and as examples of how to do cool animations with CSS.

  • Flexbox Guide

    Flexbox is a newer feature in CSS that allows creating layouts without using floats. It solves a lot of problems that used to be total headaches for developers, and it's something that is totally worth spending the time to learn. This is a good guide with useful pictures as to what all the flexbox CSS properties mean!

UI/UX

  • Typography for User Interfaces

    While you might not spend a lot of time thinking about it, typography is incredibly important to creating a usable interface and in supporting your message. While it's a little nerdy, this article is an interesting look into it, and gives you some good ways to think about typography.

  • Dieter Rams 10 Principles of Good Design

    Dieter Rams, famous for his product design done for Braun, talks about the 10 principles that he sees as common to all good design. These are fantastic guiding concepts that you can use to evalute your websites, as well as to think critically about the products you see around you.
